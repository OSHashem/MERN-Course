
# Day 5: JavaScript Functions and Temperature Conversion

## Overview
This day covers JavaScript functions, variable scope, and a practical temperature conversion program.

## Key Concepts

### Functions
- **Definition**: Reusable sections of code declared once and called multiple times
- **Parameters**: Named variables that accept values when the function is called
- **Arguments**: Actual values passed to the function
- **Return Statement**: Sends a value back from the function to the caller

### Variable Scope
- **Global Scope**: Variables accessible throughout the entire program
- **Local Scope**: Variables only accessible within their function
- Scope precedence: Functions check local scope first, then global scope
- Same variable names can exist in different scopes without conflict
