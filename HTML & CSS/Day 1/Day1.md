# Day 1: HTML Basics Summary

## Overview
Introduction to HTML fundamentals including headings, links, media, and document structure.

## Key Topics Covered

### Text Elements
- **Headings**: h1(big) through h6(small) tags for hierarchical content
- **Paragraphs**: `<p>` tags for text blocks
- **Line breaks**: `<br>` tag
- **Horizontal rules**: `<hr>` tag

### Hyperlinks
- External links with `target="_blank"` for new tabs
- `mailto:` links for email
- `title` attribute for tooltips

### Media
- **Images**: `<img>` tags with `alt` text and height attributes
- **Audio**: `<audio>` tag with controls, mute, and loop options
    - Supports mp3 and wav formats
- **Video**: `<video>` tag with playback controls
    - Supports mp4 format

### Document Setup
- Favicon implementation with `<link>` tag
- Title and meta information in `<head>`
- Proper HTML5 document structure
