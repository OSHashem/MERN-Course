### Text Formatting
- **Basic tags**: `<b>` (bold), `<i>` (italic), `<u>` (underline), `<del>` (deleted)
- **Size tags**: `<big>`, `<small>`, `<sub>` (subscript), `<sup>` (superscript)
- **Other**: `<tt>` (monospaced), `<mark>` (highlighted text)

### Span vs Div
- **`<span>`**: Inline container for styling specific text portions
- **`<div>`**: Block container that takes full width for grouping elements

### Lists
- **Unordered lists** (`<ul>`) with nested items
- **Ordered lists** (`<ol>`) for sequential content
- **Description lists** (`<dl>`, `<dt>`, `<dd>`) for term-definition pairs

### Tables
- Use `<table>`, `<tr>`, `<th>`, `<td>` for structured data
- Supports borders, alignment, and styling

### Forms
Input types: text, password, email, tel, date, number, radio, checkbox, file
- `<select>` for dropdowns
- `<textarea>` for multi-line text
- Form attributes: `action`, `method`, `enctype`

### Interactive Elements
- `<button>` with `onclick` events
- JavaScript integration for DOM manipulation



### Header & Footer

Demonstrates semantic HTML layout:
- **`<header>`**: Navigation and branding
- **`<main>`**: Primary content with image and text
- **`<footer>`**: Copyright and contact info

Uses internal styling and basic semantic structure.
