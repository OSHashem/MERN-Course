CSS Colors

There are many ways to specify colors in CSS, including:
  • Named Colors: e.g., darkblue
  • Hexadecimal Colors: e.g., #FF0000 for red
  • RGB Colors: e.g., rgb(0, 121, 0) for dark green
  • HSL (Hue, Saturation, Lightness) Colors: e.g., hsl(276, 100%, 30%) for dark purple

CSS Fonts

  • Font Family: Impact , font-size: 16px(deafult)
  • Font Family: Segoe UI , font-size: 1em (100% of the parent element's font size)
  • Font Family: Lucida Console , font-weight: bold
  • Font Family: Franklin Gothic Medium , font-style: italic

Borders

 You can define borders in css like this: 
        border: 3px solid hsl(262, 100%, 56%);
        border-radius: 20px;
 You can define borders in css like this: 
        border-style: 3px  ;
        border-width: solid;
        border-color: hsl(262, 100%, 56%)
        border-radius: 20px;
 You can define borders in css like this: 
        border-bottom: 3px solid red;
        border-top: 3px solid orange;
        border-left: 3px dotted yellow;
        border-right: 3px dotted green;
        border-radius: 10px;
